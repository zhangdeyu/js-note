# JavaScript数据类型和变量存储

## 变量与常量
### 声明
在JavaScript中，有三种声明方式，分别为
- var 声明一个变量
- let 声明一个变量
- const 声明一个常量

#### 命名规则
- 以字母、下划线、$开头
- 可以包含字母数字、下划线、$
- 不能使用保留字或者关键字
#### 变量
- var 生成的变量
    - 存在函数的作用域
    - 无块级作用域
    - 存在变量的提升问题
- let为ES6规范中声明变量的方式
    - 存在块级作用域
    - 不存在变量提升的问题
    - 变量在同一作用域中不能被重复声明。
